git 常用命令

Posted on 2021-09-25

一、撤销操作

（1）撤销工作区修改

git checkout — 文件名
    
git checkout . （撤销所有修改）

（2）撤销暂存区的add

git reset HEAD 文件名（如果不跟文件名就是撤销上次全部add）

（3）撤销commit

git reset —hard commitId

（4）取消跟踪某个文件

git rm —cached 文件名

（5）清除未跟踪文件夹

git clean -n: 查看将要被删除的文件，并不实际删除文件
    
git clean -f: 删除当前工作面临下的未跟踪文件，但不删除文件夹
    
git clean -df: 删除当前工作面临下的未跟踪文件以及文件夹

二、分支

（1）删除远程分支

git push [远程名]：[远程分支名]

（2）查看本地、远程分支

git branch git branch -a

（3）推送本地分支到远程

git push origin 分支名

（4）关联分支

git branch —set-upstream-to=origin/远程分支名  本地分支名

（5）合并分支

git merge 被合并的分支名

（6）推送本地新分支到远程并追踪（远程当前并不存在该分支）

git push -u origin 分支名

三、追踪

（1）删除未追踪的文件

删除 untracked files

git clean -f

连 untracked 的目录也一起删掉

git clean -fd

连 gitignore 的untrack 文件/目录也一起删掉 （慎用，一般这个是用来删掉编译出来的 .o之类的文件用的）

git clean -xfd

在用上述 git clean 前，墙裂建议加上 -n 参数来先看看会删掉哪些文件，防止重要文件被误删

git clean -nxfd

git clean -nf

git clean -nfd

（2）解除追踪已追踪的文件

git rm –cached 文件名 （如果是目录名，后面跟个-r）
